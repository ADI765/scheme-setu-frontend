# Scheme Setu — Frontend

Frontend for Scheme Setu (EdTech Government Scheme Discovery Platform)

## Tech Stack
- HTML
- Tailwind CSS (CDN)
- Vanilla JavaScript

## How to Run

1. Open with Live Server (VS Code extension)
OR
2. Open index.html directly in browser

## Current Mode

The project is using dummy data.

To connect backend:

Open js/api.js

Change:
const USE_DUMMY = true;

To:
const USE_DUMMY = false;

Backend endpoint expected:
POST /api/match-schemes
